📌 Project Management Application

A modern Project Management system that allows users to create, edit, and manage projects, along with their targets, tasks, and notes efficiently.

🚀 Features
📁 Project Management
Create new projects
Edit existing projects
View project details
Delete projects (optional)

🎯 Target Management
Define goals/targets for each project
Track progress of targets
Set deadlines

✅ Task Management
Add tasks under targets/projects
Update task status (Pending / In Progress / Completed)
Assign priorities
Edit and delete tasks

📝 Notes
Add notes to projects or tasks
Edit notes anytime
Useful for documentation and reminders

🧩 Core Functionalities
1. Create Project
Add project title, description
Set start & end date
Attach targets, tasks, and notes
2. Edit Project
Update project details
Modify targets and tasks
Add or update notes
3. View Project
View complete project overview
Track targets and task progress
Read associated notes

File attachments
Role-based access control
